## API интерфейс

> Заполни если есть REST API, gRPC, или любой другой межсервисный интерфейс.

### Базовый URL

```
http://localhost:8000/api/v1
```

### Аутентификация

```
Authorization: Bearer <token>
```

### Эндпоинты

#### [Модуль — например: Auth]

```
POST /auth/register
POST /auth/login
POST /auth/logout
POST /auth/refresh
```

**POST /auth/login**

Request:
```json
{
  "email": "user@example.com",
  "password": "..."
}
```

Response 200:
```json
{
  "access_token": "...",
  "refresh_token": "...",
  "expires_in": 86400
}
```

Response 401:
```json
{
  "error": "invalid_credentials"
}
```

#### [Следующий модуль]

```
GET    /resource          # список
POST   /resource          # создать
GET    /resource/:id      # получить
PUT    /resource/:id      # обновить полностью
PATCH  /resource/:id      # обновить частично
DELETE /resource/:id      # удалить
```
